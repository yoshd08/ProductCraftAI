# **App Name**: StudyWise

## Core Features:

- Input Handling: Allows users to input case study text either by uploading a file (e.g., .txt, .docx, or .pdf) or pasting text into a rich text editor.
- Framework Engine: Leverages an AI tool to analyze case study text against definitions of 5 user-selected Product Management Frameworks, extracting relevant insights for each framework.
- Results Dashboard: Presents a results dashboard that includes the framework name, a key insight extracted from the case study for each framework, and a suggested strategy or improvement.
- Framework selection: Lets the user to choose any five product management frameworks from a pre-populated list.

## Style Guidelines:

- Primary color: Muted blue (#6699CC), providing a professional and calm feel.
- Background color: Very light blue (#E6EEF5), for a clean and unobtrusive backdrop.
- Accent color: Light desaturated blue (#77AADD), to highlight key interactive elements without being distracting.
- Headline font: 'Space Grotesk', a sans-serif for headlines and short amounts of body text; body font: 'Inter', a sans-serif for longer passages.
- Dashboard layout with a left or top navigation bar, a main analysis panel, and framework insights displayed in responsive cards.
